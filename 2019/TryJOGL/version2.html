Version 2：OpenGLのキャンバスをフレームの中身として入れる

今回はJavaとOpenGLをくっつけるために「Version 1」を「Version 2」にしましょう。「tryjogl」ディレクトリ（パッケージ）にある「Example.java」の中身を、以下のソースコードと同じになるように直してください。編集するところをハイライトで表しておきます。Swingのフレーム（ウィンドウ）の中にOpenGLのキャンバスを配置するプログラミングになります。

ソースコード（Version 2）

package tryjogl;

import com.jogamp.opengl.awt.GLCanvas;
import com.jogamp.opengl.GLCapabilities;
import com.jogamp.opengl.GLProfile;

import javax.swing.JFrame;

import java.awt.event.WindowAdapter;
import java.awt.event.WindowEvent;

/**
 * JOGL（Java bindings for OpenGL）の例題プログラムである。
 * OpenGLを使ったJavaによる三次元グラフィックスの初歩的な練習になる。
 */
public class Example extends WindowAdapter
{
    /**
     * Exampleのコンストラクタである。
     */
    public Example()
    {
        System.out.println("Example");

        // OpenGL描画のためのキャンバスを生成する。
        GLProfile profile = GLProfile.get(GLProfile.GL2);
        GLCapabilities capabilities = new GLCapabilities(profile);
        capabilities.setDoubleBuffered(true);
        GLCanvas aCanvas = new GLCanvas(capabilities);

        // フレームを生成し、上記のキャンバスを乗せて、ウィンドウとして開く。
        JFrame aFrame = new JFrame("Wire Teapot");
        aFrame.add(aCanvas);
        aFrame.addNotify();
        int titleBarHeight = aFrame.getInsets().top;
        aFrame.setSize(400, 400 + titleBarHeight);
        aFrame.setLocationRelativeTo(null);
        // aFrame.setDefaultCloseOperation(JFrame.DO_NOTHING_ON_CLOSE);
        aFrame.addWindowListener(this);
        aFrame.setFocusable(false);
        aFrame.setVisible(true);

        return;
    }

    /**
     * ウィンドウを閉じようとした時に呼び出される。
     * @param aWindowEvent ウィンドウイベント
     */
    public void windowClosing(WindowEvent aWindowEvent)
    {
        // System.out.println(aWindowEvent.paramString());

        // 終了する。
        System.exit(0);

        return;
    }

    /**
     * 針金細工のティーポット（Wire Teapot）を描くOpenGLのウィンドウを開く。
     * @param arguments コマンドの引数列（文字列の配列）
     */
    public static void main(String[] arguments)
    {
        System.out.println("main");

        new Example();

        return;
    }
}

プログラムの解説

まず、OpenGL描画のためのキャンバスであるGLキャンバス「GLCanvas」をインポートします。このGLキャンバスのインスタンスを得る正式な方法は、GLプロフィール「GLProfile」からGLケイパビリティーズ「GLCapabilities」を作り、そのGLケイパビリティーズからGLキャンバスを生成することですから、それらもインポートしなければなりません。

JOGLのバージョンが2以降になってから、GLはGL2になりました。多くの定数（リテラル）もGLの下ではなく、GL2の下にあります。そこで、引数に文字列リテラル「GLProfile.GL2」を与えたメッセージ「get(String)」を発信し、GLプロフィールのスタティックメソッド（クラスメソッド）を動かして、GL2用のGLプロフィールのインスタンスを得ます。

そして、そのGLプロフィールを引数にしてGLケイパビリティーズのインスタンスを生成し、ダブルバッファ方式の描画（チラつきを抑えて描く常套手段：いま見えているキャンバスをクリアしてイチイチ描くからチラつくのであって、見えないキャンバスを用意して、そのキャンバスに裏でシコシコ描き、描き終わったところで、いま見えているキャンバスと一気に入れ替える方式）を行うことを「setDoubleBuffered(boolean)」のメッセージで伝えておきます。

その後、そのGLケイパビリティーズを引数にしてGLキャンバスのインスタンスを獲得します。GLキャンバスはクラス階層を調べると、java.awt.Canvasのサブクラスであり、コンポーネントの一種です。ですから、Swingのフレーム（ウィンドウ）の中身に指定することが可能です。JFrameのインスタンスをレシーバにして「add(Component)」のメッセージを発信できます。

蛇足になりますが、「aFrame.setDefaultCloseOperation(JFrame.DO_NOTHING_ON_CLOSE)」をコメントアウトして、ExampleをWindowAdapterのサブクラスにすることで、クローズイベント（ウィンドウを閉じた事象）を、「aFrame.addWindowListener(this)」で受け取るように指示し、メソッド「public void windowClosing(WindowEvent aWindowEvent)」で処理（終了：「System.exit(0)」）するようになっています。
